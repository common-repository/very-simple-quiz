== Description ==

This is very simple Quizz program where you can create Quiz and show it on you page or post using shortcode.  
You can create filters to show result page based on user scores in the quiz.

This is a brief introductory video for Very Simple Quiz:

[youtube http://www.youtube.com/watch?v=9qSAWZpHx8Q]


**Features**

* Set Unlimited Question
* Assign Numbers on Two options
* Show Page based on Different results


== Installation ==

To do a new installation of the plugin, please follow these steps

1. Download the very-simple-quiz.zip file to your computer.
1. Unzip the file 
1. Upload 'readmore-readless' folder to the '/wp-content/plugins/'' directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Does this works with all themes =

This plugin has been tested with Twenty Twelve, Twenty Thirteen and Twenty Fourteen theme and the plugin works like a charm with these themes.
